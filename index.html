This viewer was inspired by the video A Better Way To Picture Atoms by Henri Reich and his team from minutephysics.

The sampling of the wave function follows the Metropolis algorithm and does not require to precompute the set of points from the cdf.

Consequently, this version does not require extra data and can be computed for any wave function which pdf can be evaluated.

Some shader functions were taken from various places on Shadertoy:

  • matplotlib colormaps + turbo
  • HSV and HSL colormaps
  • Twilight and coolwarm colormap
